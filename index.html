Projet Paint

Le projet paint est un projet qui a été réalisé après 1 mois et demi après le début des cours, il consistait à recréer un site dans lequel nous pouvons générer : des carrés, des ronds, des triangles et du texte.
J'ai rajouté des options afin de pouvoir changer :

  • La taille
  • L'orientation
  • Les couleurs
  • La police d'écriture du texte

Ce projet a atteint ce stade de développement en 3 jours.

Projet Api

Le projet Api faisant appel à une Api d'animes et permettant de faire une recherche à l'aide de mots clés et d'afficher :

  • Les images
  • Les synopsis
  • Les liens vers un site fournissant plus d'informations

Ce site a été réalisé en une journée dans le cadre d'un tp.
